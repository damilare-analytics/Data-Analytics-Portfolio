# Order Trend (Monthly)

## Project Overview

This project examines how order volume changes over time.

---

## Objective

To identify patterns in customer purchasing behavior across different months.

---

## Tools Used

* SQL (BigQuery)
* Excel

---

## SQL Approach

The analysis involved:

* Aggregating total orders by month
* Organizing results chronologically

---

## Visualization

A line chart was used to illustrate monthly order trends.

---

## Key Insight

Order fluctuations suggest possible seasonal or demand-driven patterns.

---

## Dataset

The analysis was conducted using the **thelook_ecommerce** public dataset from Google BigQuery.

This dataset simulates real-world e-commerce operations and includes transactional business data.
