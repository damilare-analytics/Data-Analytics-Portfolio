# Top 10 Products by Revenue

## Project Overview

This project identifies products generating the highest revenue.

---

## Objective

To determine top-performing products.

---

## Tools Used

* SQL (BigQuery)
* Excel

---

## SQL Approach

Products were ranked based on total revenue generated.

---

## Visualization

A bar chart was used to display top products.

---

## Key Insight

Revenue is concentrated among a few high-performing products.

---

## Dataset

The analysis was conducted using the **thelook_ecommerce** public dataset from Google BigQuery.
