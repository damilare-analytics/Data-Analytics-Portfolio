# Top 10 Customers by Total Spend

## Project Overview

This project identifies customers with the highest spending.

---

## Objective

To determine which customers contribute most to total revenue.

---

## Tools Used

* SQL (BigQuery)
* Excel

---

## SQL Approach

Customers were ranked based on total purchase value.

---

## Visualization

A bar chart was created to highlight top customers.

---

## Key Insight

A small number of customers contribute significantly to revenue.

---

## Dataset

The analysis was conducted using the **thelook_ecommerce** public dataset from Google BigQuery.
