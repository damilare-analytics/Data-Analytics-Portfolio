
# Revenue Trend by Months

## Project Overview

This project tracks how revenue changes month by month.

---

## Objective

To evaluate financial performance trends over time.

---

## Tools Used

* SQL (BigQuery)
* Excel

---

## SQL Approach

Monthly revenue was calculated by grouping transactions by month.

---

## Visualization

A line chart was used to show revenue trends.

---

## Key Insight

Revenue movement reflects overall business performance across time.

---

## Dataset

The analysis was conducted using the **thelook_ecommerce** public dataset from Google BigQuery.
